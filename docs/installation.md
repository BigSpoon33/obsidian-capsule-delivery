# Installation

## Stable release

To install Obsidian Capsule Delivery, run this command in your terminal:

```sh
uv add obsidian-capsule-delivery
```

Or if you prefer to use `pip`:

```sh
pip install obsidian-capsule-delivery
```

## From source

The source files for Obsidian Capsule Delivery can be downloaded from the [Github repo](https://github.com/BigSpoon33/obsidian_capsule_delivery).

You can either clone the public repository:

```sh
git clone git://github.com/BigSpoon33/obsidian_capsule_delivery
```

Or download the [tarball](https://github.com/BigSpoon33/obsidian_capsule_delivery/tarball/master):

```sh
curl -OJL https://github.com/BigSpoon33/obsidian_capsule_delivery/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd obsidian_capsule_delivery
uv pip install .
```
