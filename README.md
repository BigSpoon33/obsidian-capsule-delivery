# Obsidian Capsule Delivery System

CLI tool for AI-powered educational content generation in Obsidian.

## Installation

pip install -e .

## Basic Usage

# Show version
capsule --version

# Show help
capsule --help

## Features

• AI-powered content generation
• Template-driven capsule creation
• Educational content packaging
• Cross-domain knowledge composition
